## üöÄ Setup & Usage

1.  **Clone the repo**:
    ```bash
    git clone https://github.com/WayneChibeu/alta-weather.git
    ```

2.  **Configure API Key**:
    -   This project uses `config.js` to store the API Key safely.
    -   Rename `config.example.js` to `config.js`.

3.  **Run**:
    -   Simply open `index.html` in your browser!
